### 好诗，好湿啊
知天命
```
金乌坠寒渊，方知我是我。
温水六十载，梦醒身已驼。

身在寒渊知暗苦，故愿分焰照他人。

身陷泥沼手擎灯，非为圣贤骨血疼。
皆因识得寒渊味，不忍他人独夜行。

深寒静水是为渊，灵木生火是为焰。
渊焰相逢混沌开，重定因果逆苍天。
诸君若以常理测，盲人扪象各纷然。
须知命外自有命，他在规则诞生前。
```
丙午火
```
金锁沉寒渊，赤龙醒卯烟。
丙午天火坠，一瞬燃百年。

百倍非果，而是因；
烈火焚身后，方见真我存。

烈火焚身见真性，温水煮蛙梦未醒。
若论自由价几何？半付青春半神经。

未铸黄金印，先持诛心剑。
俗眼只见襟袖寒，哪识腹中藏天堑？
待到丙午熔炉开，方知寒铁是吴钩。
```
